That is simple REST API implementation example using Lumen Framework in PHP.
There are also migrations to create proper database.

<b>Endpoints:</b>

// registration - provides an token
POST register/

// logging in
POST login/

// sets methods
POST sets/
GET sets/
GET sets/{id}/
PUT sets/{id}/
DELETE sets/{id}/

// words methods
POST words/
GET words/{id}/
PUT words/{id}/
DELETE words/{id}/

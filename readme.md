That is simple REST API implementation example using Lumen Framework in PHP.
There are also migrations to create proper database.

<b>Endpoints:</b>

// registration - provides a token </br>
POST register/ </br>

// logging in </br>
POST login/  </br>

// sets methods </br>
POST sets/  </br>
GET sets/ </br>
GET sets/{id}/ </br>
PUT sets/{id}/  </br>
DELETE sets/{id}/ </br>

// words methods
POST words/ </br>
GET words/{id}/ </br>
PUT words/{id}/ </br>
DELETE words/{id}/ </br>
